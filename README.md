# CorkLogix - B2B Wine Platform

CorkLogix is a professional B2B wine merchant purchasing and management system. Built with a focus on visual excellence and seamless user experience, it provides a comprehensive solution for wine trading and order management.

## 🚀 Features

- **Product Catalog**: Advanced browsing with category filtering (Red, White, Rose, Sparkling).
- **B2B Order Lifecycle**: Full management of the order process from creation to payment.
- **Customer Display**: Optimized views for customer-facing interfaces.
- **Order Management**: Real-time status tracking and signing capabilities.
- **Premium Design System**: A cohesive UI/UX built with Tailwind CSS and Lucide icons.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Language**: [TypeScript](https://www.typescriptlang.org/)
- **Styling**: [Tailwind CSS](https://tailwindcss.com/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Utilities**: `clsx`, `tailwind-merge`, `date-fns`

## 📦 Getting Started

### Prerequisites

- Node.js (Latest LTS recommended)
- npm or yarn

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/TravisTien/CorkLogix.git
   ```
2. Install dependencies:
   ```bash
   npm install
   ```

### Development

Run the development server:
```bash
npm run dev
```

### Build

Build the project for production:
```bash
npm run build
```
